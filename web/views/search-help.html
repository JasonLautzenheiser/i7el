Searching The Inform 7 Extensions Library

You can search the Library much like you would any other search engine. Results will be limited to those extensions which match every search term you give.

Search options

In addition to general keyword searches, you can search for extensions which match particular information through using these tags:

title:
title:glulx
title:"glulx text effects"
author:
author:emily
author:"Emily Short"
tag:
tag:actions
tag:glulx

These tagged terms can be combined with each other and with regular terms:

title:vorple images
title:glulx author:emily
